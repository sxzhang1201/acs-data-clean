# Maintainers

This repository is hosted under the `nmcb-fair` GitHub organization so the project can remain available after individual contributors move roles.

## Current Maintainer

- Shuxin Zhang, original author

## Future Maintenance

Before Shuxin Zhang leaves the project, the `nmcb-fair` organization should assign at least one continuing maintainer with write or maintain access.

Maintainers are responsible for:

- Reviewing issues and pull requests.
- Creating releases.
- Keeping documentation clear for non-technical ACS users.
- Making sure no real participant data is uploaded to the repository, issues, or pull requests.

## Access Recommendation

The original author can remain as an outside collaborator if the organization allows it. This preserves continuity without making one person the only owner of the repository.
